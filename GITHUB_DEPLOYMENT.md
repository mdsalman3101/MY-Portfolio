### ⚙️ STEP 4: Enable GitHub Pages (2 minutes)

## ⚠️ TROUBLESHOOTING

### Issue: Portfolio doesn't load

**Solution:**
1. Check the URL is exactly correct
2. Wait 5-10 minutes (might be building)
3. Hard refresh browser (Ctrl+Shift+R)
4. Check that files are in main branch

---

### Issue: Photo not showing

**Solution:**
1. Make sure photo is named exactly `profile.jpg`
2. Hard refresh (Ctrl+Shift+R)
3. Check file is uploaded to GitHub

---

### Issue: Styling looks broken

**Solution:**
1. Wait for GitHub to build (2-3 minutes)
2. Hard refresh (Ctrl+Shift+R)
3. Check all files are uploaded correctly

---

### Issue: Changes not appearing

**Solution:**
1. Wait 3-5 minutes for GitHub to update
2. Hard refresh (Ctrl+Shift+R)
3. Check you committed changes
4. Verify you're looking at correct branch (main)

---

### Issue: 404 Error Page

**Solution:**
1. Repository is not public - Fix: Go to Settings → Change visibility to Public
2. GitHub Pages not enabled - Fix: Go to Settings → Pages → Enable it
3. Files not in root folder - Fix: Upload files to root (not in subfolder)
