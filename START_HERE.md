## 🔧 CUSTOMIZATION QUICK REFERENCE

### **Easy Changes**

**Change your name:**
Find: `<h1 class="hero-title">MD Salman</h1>`
Change to: `<h1 class="hero-title">Your Name</h1>`

**Change colors:**
Find: `--accent-cyan: #00d4ff;`
Change to: `--accent-cyan: #00ff88;` (for green, for example)

**Add projects:**
Copy the project-card section and paste multiple times

**Update skills:**
Copy skill-item blocks and change skill names

**Change email/phone:**
Search for your current email/phone and replace

### **How to Make Edits**

1. Open file in **Notepad** or **VS Code**
2. Find what you want to change
3. Make the change
4. Save (Ctrl+S)

## ❓ COMMON QUESTIONS

### **Q: Is it free?**
A: Yes! GitHub Pages is completely free. No credit card needed.

### **Q: Can I change my portfolio later?**
A: Yes! Edit on GitHub directly or download, edit, and reupload.

### **Q: How long until Google indexes me?**
A: Usually 1-2 weeks. You can request it in Search Console to speed up.

### **Q: Can I use a custom domain?**
A: Yes! GitHub Pages supports custom domains (yourdomain.com instead of github.io)

### **Q: What if I break something?**
A: You can restore files from GitHub. Use Ctrl+Z in your editor. Don't worry!

### **Q: How do I add more projects?**
A: Copy the project-card section in HTML and modify it with your project details.

### **Q: Can I change the colors?**
A: Yes! Edit the CSS variables at the top of styles.css.
